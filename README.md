
Video Player using XState
Project Description
A mini web application video player implemented using the "Finite State Machines" pattern based on the XState library. The application demonstrates video player state management through a finite state machine with the ability to switch between fullscreen and minimized modes.

Functional Requirements
Core Functionality
✅ Video playback from a specified source

✅ Playback control (play/pause)

✅ Switching between fullscreen and minimized (picture-in-picture) modes

✅ Closing the video player

✅ State management through the XState finite state machine

Видеоплеер с использованием XState
Описание проекта
Мини веб-приложение видеоплеера, реализованное с использованием паттерна "Конечные автоматы" (State Machines) на базе библиотеки XState. Приложение демонстрирует управление состояниями видеоплеера через конечный автомат с возможностью переключения между полноэкранным и уменьшенным режимами.

Функциональные требования
Основной функционал
✅ Воспроизведение видео из заданного источника

✅ Управление воспроизведением (play/pause)

✅ Переключение между полноэкранным и уменьшенным (picture-in-picture) режимами

✅ Закрытие видеоплеера

✅ Управление состоянием через конечный автомат XState
